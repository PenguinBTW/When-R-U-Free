# When-R-U-Free
A simple app designed to help you fine when friends at sixth form / college / uni are free to hangout between lessons. This is designed to make it much easier to study within these breaks due to the scattered nature of timetables.

### Features
**MVP**
* Basic ui to display when friends are free
* Basic backend to store data imputted into app (start with firebase)
* Tech stack: Flutter (frontend), Node js (server backend), (firebase auth + data storage)
* UI to input your timetable
* Backend logic to compare friends timetable to yours

**Secondary Goal**
* Automatic sync with microsoft graph api with a no data stored architecture (only refresh key)
